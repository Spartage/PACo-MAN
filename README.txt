Para correr el programa, primero se necesita tener instalada la libreria pygame, lo cual se puede hacer con el comando
'pip install pygame' por consola. luego dirigirse a la carpeta donde esta guardado el archivo Game.py con el comando 
'cd {Direccion de carpeta}' sin los corchetes. Estando ya en la carpeta con la consola de comandos, escribimos 'python Game.py'
y ya estaremos corriendo el juego.
